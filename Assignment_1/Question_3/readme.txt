Print Numbers from 5 to 1 Using a Loop
--------------------------------------

Description:
This program prints the numbers from 5 down to 1 in reverse order.
A loop is used to handle the repetition and countdown.
There is no user input required.

Behavior:
- The program displays the numbers 5, 4, 3, 2, and 1, each on a new line.
- A loop runs in reverse to perform the countdown.

Example Output:
5
4
3
2
1

Instructions:
1. Open the source file in your editor.
2. Compile the program using your preferred method.
3. Run the compiled output in the terminal or command line.
