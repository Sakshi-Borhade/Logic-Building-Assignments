Division of Two Numbers
-----------------------

Description:
This program performs division of two whole numbers.
The values for the dividend and divisor are set within the program itself.
No input is required from the user when running the program.

Behavior:
- The result is calculated using whole number division (any remainder is discarded).
- If the divisor is zero, an error message is displayed to prevent invalid division.

Example:
Given:
  dividend = 10
  divisor = 3

Output:
  Result: 3

Instructions:
1. Compile the program using your preferred compiler.
2. Run the compiled output from the terminal or command line.

Note:
Modify the values of the dividend and divisor in the source file to change the inputs.
